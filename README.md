# AI Disaster Response Assistant
Official repository for the IBM Hackathon.

## 🎯 Target Demo Scenario (Day 5 Goal)
Earthquake Trigger -> Share Location -> Chat with AI -> View Map -> View Emergency Kit -> Switch to Hindi

## 📂 Project Structure
- `/frontend`: Next.js & Tailwind CSS
- `/backend`: FastAPI & Supabase
- `/ai_module`: IBM Granite integration
- `/maps_module`: Leaflet & OpenStreetMap API
- `/qa_docs`: Testing logs and demo scripts
